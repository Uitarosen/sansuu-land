# 小3ワールド仕様「おかし工場」🏭

3年生は「かけ算・わり算の完成」と「小数・分数のはじめて」の学年。
抽象度が一気に上がるため、**操作型(手で分ける・測る)を最も厚く**配置する。

## 1. 世界観

ベルトコンベアを流れるおかしを、数え、仕分け(わり算)、箱詰め(かけ算)、検品(重さ・長さ)して出荷する工場。
単元マスターごとに工場に機械が1台ずつ増え、ライン稼働アニメが賑やかになる。相棒はロボットのカカオくん。

## 3. 山場単元と攻略設計

### (a) わり算の導入(3-3)— 学年最重要
- **ShareBoard 操作型**: 「クッキー12こを 3まいの さらに おなじかずずつ」→ タップで1個ずつ皿に配る(等分除)。「4こずつ はこに つめると なんはこ?」→ 4個ずつ束ねる(包含除)
- 等分除と包含除を**同じ数値で対比出題**し、どちらも 12÷3 になることを発見させる
- 九九の逆(「3×□=12」)との接続を kuku 型で混在出題

### (b) あまりのあるわり算(3-5)
- quotRem 2欄回答(「7 あまり 1」)
- 回答が「あまり ≧ わる数」のとき専用ヒント「まだ わけられるよ!」を即時表示
- 検算レッスン(わる数×商+あまり=わられる数)を必ず入れる
- テスト制約: 生成問題は必ず あまり < わる数

### (c) 時間の計算(3-2)— 60進法のつまずき
- NumberLine を時間軸にした「時間ライン」表示: 9:40→10:10 は「10:00 で区切って 20分+10分」と視覚化
- 「1分=60秒」「短い時間はストップウォッチ」を体感クイズで

### (d) 小数・分数のはじめて(3-10, 3-12)
- 小数: 数直線タップ「0.7 は どこ?」(0.1刻み)。1Lます表示で「0.3L」も
- 分数: 「1mのテープを4等分した1つ分=1/4m」— **量としての分数**を最優先し、記号操作(同分母たし引き)は単元後半
- 小数と分数の橋渡し(0.1 = 1/10)を最終レッスンに

- 機械コレクション(単元マスターで工場に機械が増える)

- カカオくんが正解時に工場アクション(アームでおかしを掴む等)の小アニメ
