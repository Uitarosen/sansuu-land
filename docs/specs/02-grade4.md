# 小4ワールド仕様「スイーツ遊園地」🎡

4年生は「わり算の筆算」という小学算数最大級の手続き学習と、「角度・面積・展開図」の空間認識が柱。
**手続きはステップ分解+弱点記録**、**空間は必ず動かして見せる**が設計方針。

## 1. 世界観

アトラクションがそのまま単元になっている遊園地。観覧車で角度、ジェットコースターで折れ線グラフ、園内マップで面積。
単元マスターでアトラクションが開園し、全開園で花火ショー演出。相棒はペンギンのソフトくん。

## 3. 山場単元と攻略設計

### (a) わり算の筆算(4-3 / 4-6)— 学年最大の山場
- **LongDivisionBoard**: 「たてる → かける → ひく → おろす」を1ステップずつガイド付きで入力。各ステップが小問になっており、どこで間違えたかが特定できる
- **subSkill をステップ別に記録**(div-tateru / div-kakeru / div-hiku / div-orosu)し、弱いステップを含む問題を重点出題(九九ヒートマップと同じ仕組み)
- 慣れたらガイドを外した通常入力(答えのみ)に自動昇格(difficulty 3)
- 商に 0 が立つ問題(816÷4=204)を必ず一定割合で出題(0落ち対策)

### (b) 分度器(4-4)
- **Protractor**: ドラッグで分度器を回して辺に合わせ、目盛りを読む。中心合わせ→0°線合わせ→読む、の3ステップをガイド
- 誤答には必ず **180−θ(内側・外側目盛りの読み違い)** を含める
- 導入レッスンは「かどの開き=角度」を扇が開くアニメで(長さではなく開き、を強調)

### (c) がい数(4-7)
- 「**どの位で**四捨五入するか」を最初に問う2段階出題
- NumberLine で 2,347 が 2,000 と 3,000 のどちらに近いかを見せてから手続き化
- 「約」の使いどころ(見積もり)を買い物場面の文章題で

### (d) 展開図(4-14)
- **UnfoldedBox**: 展開図候補から「立方体になるもの」をタップ → 正誤どちらでも折りたたみアニメで検証(間違いも折って「重なる/足りない」を見せる)
- 面の数・重なる辺の対応をタップで答える設問も
