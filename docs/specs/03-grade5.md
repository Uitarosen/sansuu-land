# 小5ワールド仕様「スイーツ研究所」🔬

5年生は小学算数の分水嶺。「割合」「通分」「単位量あたり」でつまずく子が最も多い。
**割合と分数は操作型に最大投資**し、トーンも「実験・研究」で知的に寄せる(パステル→ミント/ネイビー)。

## 1. 世界観

白衣のみみちゃんとふくろうのドクターオウルが働く研究所。濃度実験=割合、水そう=体積、調合=通分、計測データ=平均。
単元マスターで実験器具が図鑑に増え、研究員→主任→博士とランクアップ。

## 3. 山場単元と攻略設計

### (a) 割合(5-14)— 小学算数最難関
- **RatioMeter**: 「もとにする量」を**必ず先にタップで指定**→比べる量→割合が帯メーターに表示される操作。立式より先に「どっちが『もと』か」の判断だけを訓練する専用問題を置く
- レッスンで「**『の』の前=もとにする量**」(「定員の 0.8倍」)のことば規則を徹底
- %⇔小数変換は専用ミニドリル(subSkill: percent-convert)としてヒートマップ管理
- 「くらべる量=もと×割合」の1本だけを公式とし、みはじ式の3公式暗記はさせない(残り2つは□の式で導く)

### (b) 通分・約分(5-9)
- **FractionBar**: 1/2 と 1/3 の2本のバーの分割数をボタンで増減(2,4,6…/3,6,9…)し、**目盛りがそろう瞬間(6等分)を自分で発見**する操作。公倍数の意味が視覚で腹落ちしてから手続き化
- 約分は「等価分数を見つけるゲーム」(6/8 のバーと同じ長さはどれ?)から導入
- 誤答には必ず「分母同士・分子同士を足す」(1/2+1/3=2/5)を含める

### (c) 面積公式(5-13)
- **AreaTransform**: 平行四辺形の端をドラッグで切り取り→反対側へ移動→長方形になる操作。三角形は2枚合わせ、台形は回転合体
- 「変形すれば長方形」を体験してから公式化。公式先出しのレッスンは禁止

### (d) 単位量あたり・速さ(5-11 / 5-12)
- どちらも「**1あたりにならす**」に統一した教え方。二重数直線(DoubleNumberLine)を全問題に添える
- みはじの図(は・じ・き)は採用しない — 意味理解を優先

### (e) 小数の乗除(5-3 / 5-4)
- 「×小数=増えるとは限らない」を明示的にレッスンで扱う(80×0.5 の予想→検証)
- 小数点移動は「両方を10倍しても商は同じ」実験アニメで意味づけ
