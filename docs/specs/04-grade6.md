# 小6ワールド仕様「おかし宇宙ステーション」🚀

6年生は抽象化(文字と式・比例)と統合(6年間の総まとめ)の学年。
単元を「ミッション」として演出し、**最終単元=地球帰還(卒業)** で6年間の学習が完結する物語構造にする。

## 1. 世界観

宇宙ステーションを舞台に、各単元がミッション。燃料調合=分数の乗除、航行グラフ=比例、ドッキング経路=場合の数、ミッションログ=データ分析。相棒は宇宙ねこのコメット。
全ミッション完了で地球帰還=卒業エンディング(全学年キャラ登場、卒業証書発行)。

## 3. 山場単元と攻略設計

### (a) 分数÷分数(6-4)—「ひっくり返す」の意味
- レッスン2段構え: ①「わる数を1にする」(わる数・わられる数の両方に逆数をかけても商は同じ)のステップアニメ ②面積図での確認
- 誤答には必ず「逆数にせずそのままかけた値」(2/3÷4/5 → 8/15)を含める
- 「÷小数=小さくなるとは限らない」の5年学習と接続(÷1/2 は2倍になる驚きを導入に)

### (b) 比例と反比例(6-9)
- **PlotBoard**: 表の値を読んで座標を自分でタップしてプロット→点が一直線に並ぶことを**発見**→「y=決まった数×x」へ
- 反比例は「かけて一定」(x×y=24)を表で見つけるところから。グラフは読み取りのみ(作図は中学へ)
- 文字と式(6-2)の x・y をここで実用化し、抽象の意味を回収する

### (c) 場合の数(6-10)
- **TreeDiagram**: 枝をタップで伸ばして全パターンを数え上げる→「計算でも求められそう」への橋渡し
- 並べ方(順序あり)と組み合わせ(順序なし)を**同じ題材**(3人から2人選ぶ/並べる)で対比出題し、違いを際立たせる
- 落ち・重なりを防ぐ「順序よく書き出す」習慣をヒントで指導

### (d) 対称な図形(6-1)
- **SymmetrySheet**: 方眼上で対称の軸(または対称の中心)を基準に、対応する点をタップして図形の残り半分を完成させる作図操作
- 線対称と点対称を同一図形で判定するクイズ(両方/片方/どちらでもない)

### (e) 円の面積(6-7)
- AreaTransform 円版: 扇形に切って交互に並べ替え→平行四辺形に近づくアニメ操作 → 縦=半径・横=円周の半分 → 公式化
- 誤答には「直径×直径×3.14」と「円周(半径×2×3.14)との混同」を含める

### (f) 地球帰還ミッション(6-13)— 総復習モード
- 全学年の wrongHistory と理解度スコアから**弱点を自動編成**する10問×3ステージ(離陸準備→大気圏→着陸)
- クリアで卒業証書+エンディング。クリア後も「もう一度」で何度でも編成し直せる(間隔反復の受け皿)
