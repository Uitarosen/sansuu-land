# さんすうランド ✿

小学1・2年生向け 算数学習Webアプリ（仕様書 v2.0 の実装）。
「まなぶ（概念学習）」と「れんしゅう（演習）」の2モード＋マスタリーラーニングを中核に、
全30単元（小1×12 / 小2×18）をカバーします。

## 使い方

```bash
npm install
npm run dev      # 開発サーバー（http://localhost:5173）
npm run build    # 本番ビルド（dist/）
npm run preview  # ビルド結果のプレビュー
```

インストール不要・URLアクセスのみ・完全クライアントサイド（バックエンド不要）。
学習データは localStorage に端末内保存され、外部送信は一切ありません。

## 実装済みの機能（仕様書対応）

| 仕様 | 実装 |
|------|------|
| §2 学習単元 | 小1「おかしの国」12単元 ＋ 小2「スイーツのお城」18単元 をJSON的にデータ駆動で定義 |
| §3.1 たんげんマップ | ワールド切替タブ、進捗表示（未学習/学習中🌱/マスター👑）、下部ナビ |
| §3.2 まなぶ | 紙芝居レッスン、進捗ドット、音声読み上げ、たしかめクイズ、スタンプ付与 |
| §3.3 れんしゅう | 10問セッション、選択/テンキー/ひっ算/九九/時計/図形/テープ図/グラフ/単位、褒め・ヒント・解説の段階フィードバック |
| §3.4 マスタリー | 理解度スコア0〜100、難易度自動昇降（30/70）、弱点加重出題、マスター判定、九九の段別サブマスタリー、前提単元リンクによる復習導線 |
| §3.5 ごほうび | スター、きせかえ（購入・装備）、九九マスターかんむり |
| §3.6 保護者機能 | チャイルドロック、単元別理解度、九九ヒートマップ、設定、リセット |
| §4 デザイン | パステルスイーツ配色、丸ゴシック、キャラ（みみちゃん＋ショコラ）、紙吹雪・ぷにぷに演出 |
| §4.3 サウンド | WebAudio合成の効果音、Web Speech APIの日本語読み上げ |
| §6 技術 | React 18 + Vite + Tailwind + Zustand(persist) + Framer Motion |

## 構成

```
src/
├── data/        単元定義（grade1 / grade2 / items）
├── engine/      問題ジェネレータ・出題選択・採点（generators / session / rng）
├── store/       Zustand 永続ストア（理解度・スター・カレンダー）
├── lib/         speech / sound / date
├── components/  common・figures（SVG図形/時計/筆算/グラフ等）・answer（入力UI）
└── screens/     Title / GradeSelect / Home / ModeSelect / Lesson / Practice / Result / Kuku / Kisekae / Calendar / Parent
```
